## 🚀 如何使用

### 启动项目
```bash
cd /Users/mona/hermes-dashboard
./start.sh
```

### 手机访问
1. 确保手机和电脑在同一 WiFi
2. 获取电脑 IP：`ipconfig getifaddr en0`
3. 手机浏览器访问：`http://[IP]:10007`

### 安装为 App
- iOS：Safari → 分享 → 添加到主屏幕
- Android：Chrome → 菜单 → 安装应用
